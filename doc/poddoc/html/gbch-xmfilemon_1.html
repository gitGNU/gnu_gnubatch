NAME

gbch-xmfilemon - X/Motif file monitor interface

SYNOPSIS

gbch-xmfilemon &

DESCRIPTION

Gbch-xmfilemon is a simple dialog interface to set up parameters for gbch-filemon(1).

FILES

~/.gnubatch configuration file (home directory)

.gnubatch configuration file (current directory)

filemon.help message file

ENVIRONMENT

FILEMONCONF

path name of alternative message file.

SEE ALSO

gbch-filemon(1).
