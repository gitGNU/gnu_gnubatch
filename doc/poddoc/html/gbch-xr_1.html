NAME

gbch-xr - GTK+ batch job submission

SYNOPSIS

gbch-xr &

DESCRIPTION

Gbch-xr is a fully interactive GTK+ alternative to the standard tools for submitting batch jobs, gbch-r(1) and gbch-rr(1). Jobs are submitted from saved job files, which may have been created via "unqueue" from gbch-q(1), gbch-xq(1) or gbch-jdel(1), or created afresh within gbch-xr.

Unlike gbch-r(1) etc there are no specific command line options to gbch-xr.

FILES

~/.gnubatch configuration file (home directory)

.gnubatch configuration file (current directory)

xmbtr.help message file

SEE ALSO

gbch-q(1), gbch-r(1), gbch-user(1), gbch-jdel(1), gbch-jchange(1), gbch-jlist(1), gbch-vlist(1), gnubatch.conf(5), gnubatch.hosts(5).
