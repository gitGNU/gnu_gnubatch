NAME

gbch-atcover - emulate AT

SYNOPSIS

gbch-atcover options

DESCRIPTION

gbch-atcover may be used instead of the standard at(1) command. It converts the options that most versions of the at(1) commands take to the equivalent gbch-r(1) commandds and then invokes btr(1) to submit the batch jobs.

The gbch-r(1) program provides a much greater set of facilities than at(1). Although it is strongly recommended to switch to the gbch-r(1) command to take advantage of these, they can be made available to users of our gbch-atcover command. This is done by setting up GBCH_R in the application or user environment.

gbch-atcover may be installed in place of at(1) in /usr/bin, with the original binary moved to something like old.at in the same directory.
