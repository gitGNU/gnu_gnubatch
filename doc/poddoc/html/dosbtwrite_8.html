NAME

dosbtwrite - write messages to Windows PC

SYNOPSIS

dosbtwrite options

DESCRIPTION

Dosbtwrite sends messages to Windows PCs similar to btwrite(8) does for user's terminals in response to the equivalents of the -w options of gbch-r(1) and equivalents. This is only done for jobs which originated on Windows PCs.

The Windows PC must be running an appropriate service for this to be effectual.

If the job was submitted by a user working from a client with a DHCP-allocated IP address, a message may be received on all clients currently logged-in with that user name.

FILES

btint-config message file

ENVIRONMENT

SPHELPDIR

alternative location for help file directory.

NOTES

Dosbtwrite is identical to GNUspools's dosspwrite(8), apart from using a different message file.

SEE ALSO

btwrite(8), btmdisp(8).
