NAME

gbch-xq - GTK+ batch queue view

SYNOPSIS

gbch-xq

DESCRIPTION

gbch-xq is a fully interactive GTK+ alternative to the standard queue manager gbch-q(1). As with gbch_q(1) the format of the screen display, the help messages and the command keystrokes can be easily altered to suit your requirements.

Unlike gbch-q(1) there are no specific command line options to gbch-xq.

FILES

~/.gnubatch configuration file (home directory)

.gnubatch configuration file (current directory)

xmbtq.help message file

SEE ALSO

gbch-q(1), gbch-r(1), gbch-user(1), gbch-jdel(1), gbch-jchange(1), gbch-var, gbch-jlist(1), gbch-vlist(1).
