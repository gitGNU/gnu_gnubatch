NAME

gbch-xmr - X/Motif batch job submission

SYNOPSIS

gbch-xmr &

DESCRIPTION

Gbch-xmr is a fully interactive Motif alternative to the standard tools for submitting batch jobs, gbch-r(1) and gbch-rr(1). Jobs are submitted from saved job files, which may have been created via "unqueue" from gbch-q(1), gbch-xmq(1) or gbch-jdel(1), or created afresh within gbch-xmr.

Unlike gbch-r(1) etc there are no specific command line options to gbch-xmr. The facility to change or specify resources settings for an X11 (and hence Motif) program on the command line can be used.

FILES

~/.gnubatch configuration file (home directory)

.gnubatch configuration file (current directory)

xmbtr.help message file

ENVIRONMENT

XMBTQCONF

path name of alternative message file.
